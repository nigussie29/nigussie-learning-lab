# SkillBridge Academy

# SkillBridge Academy Teaching Standard (SATS v1.0)

> "We don't just teach students what to think.
> We teach them how to think."

---

# Purpose

The SkillBridge Academy Teaching Standard (SATS) defines the instructional framework used across every subject offered by SkillBridge Academy.

Every lesson should help students move through the SkillBridge Learning OS:

Learn
→ Practice
→ Build
→ Master
→ Apply
→ Create
→ Lead

The objective is to create independent thinkers, problem solvers, innovators, and lifelong learners.

---

# Phase 1 — Discover

Students become curious before instruction begins.

Every lesson starts with:

- Lesson Header
- Essential Question
- Big Idea
- Learning Objectives
- Warm-up / Curiosity Challenge

Goal:

Create curiosity before explanation.

---

# Phase 2 — Understand

Students build conceptual understanding.

Include:

- Concept Development
- Discovery Activity
- Multiple Representations
- Visual Explanations

Goal:

Understand WHY before HOW.

---

# Phase 3 — Practice

Students gain confidence.

Include:

- Worked Examples
- Guided Practice
- Independent Practice

Goal:

Develop mathematical fluency.

---

# Phase 4 — Think

Students become mathematical thinkers.

Include:

- Investigation
- Pattern Recognition
- Error Analysis
- Discussion Questions

Goal:

Develop reasoning.

---

# Phase 5 — Apply

Students connect learning to real life.

Include:

- Real-world Modeling
- Career Connections
- Cross-disciplinary Applications
- Beyond the Classroom

Goal:

Show the purpose of learning.

---

# Phase 6 — Build

Students create something.

Every lesson ends with a Build Challenge.

Examples:

- Mathematical Model
- Presentation
- Poster
- Programming Project
- Simulation
- Dataset
- Proof
- Research
- Video
- Portfolio Artifact

Goal:

Students create instead of only consuming information.

---

# Phase 7 — Reflect

Students evaluate their learning.

Include:

- Quick Assessment
- Reflection
- Learning Journey

Questions include:

- What did I learn?
- What challenged me?
- What can I now do?
- What comes next?

Goal:

Encourage metacognition.

# Teaching Principles

Every lesson should:

- Encourage curiosity.
- Promote conceptual understanding.
- Use multiple representations.
- Require mathematical reasoning.
- Connect mathematics to everyday life.
- Build confidence.
- Encourage discussion.
- Include reflection.
- End with creation.
